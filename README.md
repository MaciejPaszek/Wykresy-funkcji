# Wykresy funkcji
Aplikacja rysuje wykresy funkcji o zadanych parametrach.
+ Wielomian rzędu trzeciego
+ Funkcja potęgowa
+ Funkcja wykładnicza
+ Funkcja sinusoidalna

Program posiada podstawowe zabespieczenia przed błędami wprowadzania danych.
